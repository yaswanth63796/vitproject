# QuickFix Tech Support Platform

![QuickFix Logo](static/images/logo.png) <!-- Add your logo if available -->

A comprehensive service booking platform for plumbing, electrical, and tech support services.

## Features

- **User Authentication**: Secure registration and login system
- **Service Booking**: Book plumbers, electricians, and tech support
- **Service Provider Management**: Track available service providers
- **Real-time Availability**: Check nearby service providers
- **Booking Management**: View and manage your bookings

## Technologies Used

- **Backend**: Python, Flask
- **Database**: MongoDB
- **Frontend**: HTML5, CSS3, JavaScript
- **Authentication**: Werkzeug Security
- **Templating**: Jinja2

## Installation

1. **Prerequisites**:
   - Python 3.8+
   - MongoDB
   - pip

2. **Setup**:

   ```bash
   # Clone the repository
   git clone https://github.com/yourusername/quickfix-tech-support.git
   cd quickfix-tech-support

   # Create and activate virtual environment (recommended)
   python -m venv venv
   source venv/bin/activate  # On Windows use: venv\Scripts\activate

   # Install dependencies
   pip install -r requirements.txt
